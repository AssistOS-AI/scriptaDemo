# Core Narrative Structure: Echo And Rupture

## Structure Description

Later chapters mirror earlier ones while breaking their assumptions. Each observation of Origaya echoes an Oriveni principle, then ruptures it.

## Application

- Chapter 1 establishes the deepCanopy as home, ethics as biological reality → Chapter 5 ruptures this by showing that ethics depends on conditions, not nature
- Chapter 2 shows the first horror of predation → Chapter 6 ruptures this by asking whether horror is the right response or merely the local one
- Chapter 3 shows Aethon on the surface, experiencing solar intensity → Chapter 7 ruptures this by showing Caelith using that same intensity to justify transgression
- Chapter 4 shows the probe launch as intellectual exploration → Chapter 8 ruptures this by showing that exploration has consequences that cannot be contained

## Rhythm

- Each echo should feel like progress, like understanding
- Each rupture should feel like the ground shifting — not because the progress was wrong, but because it was incomplete
- The final chapter echoes the opening but with a fundamentally different understanding of the same world
