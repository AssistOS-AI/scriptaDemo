# Archetype Matrix

## Selected Constellation

- **Protagonist** -> **Witness Scribe**: Drive: Must record the truth before power edits it. Gift: Attention to nuance and contradiction. Shadow: Believes observation alone is innocence.
- **Counterpart** -> **Threshold Guide**: Drive: Controls access to hidden systems and hidden meanings. Gift: Can translate the impossible into usable action. Shadow: Uses explanation to create dependence.
- **Pressure Source** -> **Grief Bearer**: Drive: Keeps faith with what was lost. Gift: Emotional truth and memory. Shadow: Mistakes remembrance for loyalty to the dead.
- **Witness** -> **Exile Heir**: Drive: Feels obligated to restore a broken inheritance. Gift: Persistence across estrangement. Shadow: Confuses ownership with belonging.
- **Catalyst** -> **Wounded Seeker**: Drive: Needs a pattern that can make suffering legible. Gift: Curiosity that keeps moving even through fear. Shadow: Treats obsession as depth.

## Usage Rules

- Do not flatten archetypes into labels; each must create dramatic behavior.
- Keep the gift and shadow active in scene design.
- If the user supplies explicit character instructions, preserve those and use archetype language as a pressure lens rather than a replacement.
