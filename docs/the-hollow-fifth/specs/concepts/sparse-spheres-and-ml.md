# Sparse Spheres And Machine Learning

In high-dimensional geometry, most of the volume of a sphere concentrates in unintuitive regions and neighborhoods become effectively empty for many algorithms. In machine learning this sparsity undermines distance-based intuition, weakens local similarity, and makes generalization more difficult. The book treats this as the cognitive template Adrian already knows before entering the fifth-dimensional realm.

## Story Use

- Gives Adrian a credible language for what he experiences.
- Connects scientific thought to fantastical perception.
- Prepares the later inversion: what hurts learning can preserve law.
