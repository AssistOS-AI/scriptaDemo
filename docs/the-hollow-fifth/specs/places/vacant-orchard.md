# Vacant Orchard

The Vacant Orchard is filled with fruit-shaped hollows wrapped in translucent skins. Each fruit contains information arranged around an empty center, as if meaning survives by orbiting absence. The orchard should feel beautiful, mournful, and quietly predatory.

## Function In Story

- Externalizes the idea that emptiness can be structural rather than accidental.
- Brings Mira's memory into contrast with Syl's hunger.
- Reveals that Adrian himself is being treated as a resource.
